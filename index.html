üîê Repository Setup

Add the Hat Labs repository to your system:

curl -fsSL https://apt.hatlabs.fi/hat-labs-apt-key.asc | sudo gpg --dearmor -o /usr/share/keyrings/hatlabs.gpg echo "deb [signed-by=/usr/share/keyrings/hatlabs.gpg] https://apt.hatlabs.fi " | sudo tee -a /etc/apt/sources.list.d/hatlabs.list sudo apt update

Replace <distribution> with your desired distribution and <components> with available components (e.g., main hatlabs)
